# Bookbrain

App für Leser mit einem weniger ausgeprägten Namensgedächtnis.
Für Bücher mit vielen Charakteren, Beziehungen, Orten und Ereignissen.

Ziel: Charaktere verknüpfen (Beziehungen, Spezies), Ereignisse und Orte festhalten.
Mobil-first, damit man beim Lesen schnell Notizen auf dem Handy machen kann.

## Technologien
Backend: ASP.NET Entity Framework, C# 
Frontend: Angular
